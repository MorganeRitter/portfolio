Pixel Buster

PROJECT REALISED FOR THE INTEGRATION PROJECT OF THE BACHELOR THIRD YEAR

This is a dungeon-crawler game with four players for tablets and smartphones. These dungeons are composed of several rooms filled with traps and monsters.

This game has been developed with a nine-person team within a six-month period with the game engine Godot and with Elexir. Consequently, this project has required a high amount of organisation, as well as the establishment of a planning and a project manager (myself).
