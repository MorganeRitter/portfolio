Plant mesh segmentation

PROJECT REALISED FOR THE DIGITAL GEOMETRY COURSE

This project’s goal was to implement an algorithm which allows the segmentation of mesh, more particularly plant’s meshes.
Indeed, it was quite important to be able to separate the leaves from the stem, or the branch from the plant.

This project has been realised by a four-person team, and was based on several research articles, including: Decomposing Polygon Meshes for Interactive Applications from Xuetao Li, Tong Wing Woon, Tiow Seng Tan et Zhiyong Huang.
