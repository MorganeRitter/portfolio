Playcards Love Letter

PROJECT REALISED FOR THE HUMAN-MACHINE INTERFACE COURSE

IMPLEMENTATION OF LOVE LETTER, A PLAYINGCARDS GAME FROM SEIJI KANAI FOR 2 TO 4 PLAYERS WITH THE REACTJS LIBRARY AND THE MVC LARAVEL FRAMEWORK

This project has been realised by a five-person team. We divided ourselves between the different possible parts: two persons worked on the graphic interface, two were responsibles for the game rules and the artificial intelligence, while the last person dealt with the application’s architecture and with the exchanges between the backend and the frontend.

I worked for my part on the graphic interface, by creating specific pages like the “New game creation”. I also created the cards design in line with the one of the website.
